# Local machine settings (copy to local.md)

Copy this file to `local.md` in the repo root and fill in your paths. `local.md` is gitignored.

## Dyson Sphere Program

**Game root:**
`C:\Program Files (x86)\Steam\steamapps\common\Dyson Sphere Program`

**BepInEx plugins folder:**
`<game root>\BepInEx\plugins`

**BepInEx log (after first launch):**
`<game root>\BepInEx\LogOutput.log`

## Notes

- Install BepInEx before testing mods (see `references/README.md`).
- Each mod's `.csproj` has a `DSPPluginsPath` for post-build copy; point it at your install.
